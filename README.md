## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the server using the `npm start` command.
- Login using the pre-defined users in `data.js`
- Or create a new user and start shortening urls!
